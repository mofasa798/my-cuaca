# Learning Context

## Current Goal

## Current Stack
- JavaScript
- Node.js
- Express.js
- REST API
- TailwindCSS
- Vanilla JavaScript frontend

## Current Project
my-cuaca (proyek sedang berkembang dengan frontend dan backend terintegrasi)

## Progress
- Memiliki latar belakang Ilmu Komputer
- Sedang fokus menguatkan pola pikir engineering dan konsep backend
- Sudah membuat struktur backend Express modular
- Sudah menambahkan endpoint `GET /weather` dan service integrasi WeatherAPI
- Sudah implementasi caching sederhana dengan TTL
- Sudah menambahkan endpoint `GET /weather/forecast` dengan validasi parameter
- Sudah membuat frontend Vanilla JS yang berkomunikasi dengan backend

- Sudah add health check endpoint
- Sudah menambahkan UI glassmorphism modern dengan adaptive background
- Sudah menambahkan animasi transisi cuaca dan forecast cards

## Stack Frontend
- HTML + **TailwindCSS CLI** (utility-first CSS framework)
- Vanilla JavaScript dengan Fetch API
- TailwindCSS build process dengan minification
- Custom Tailwind components defined in @layer directives
- Adaptive background dan animated UI transisi

## Build & Development
- `npm run tailwind:watch` - development mode (auto-rebuild on changes)
- `npm run tailwind:build` - production build (minified CSS)
- `npm run dev` - jalankan backend Express dengan nodemon
- Separate input.css (source) dan output.css (generated)
